# Adventools 🚀

Bienvenue dans **Adventools**, votre compagnon numérique tout-en-un pour l'étude biblique, le chant et la prise de notes.

## 📋 Informations d'Installation

Si vous rencontrez des difficultés pour installer l'application sur votre téléphone Android, veuillez vérifier les points suivants :

### 1. Configuration Minimale Requise
Pour que l'application fonctionne de manière optimale, votre téléphone doit respecter ces critères :
- **Système d'exploitation** : Android 6.0 (Marshmallow) ou une version supérieure.
- **Espace de stockage** : Au moins 150 Mo de libre (plus si vous enregistrez beaucoup de notes vocales ou d'images).
- **Mémoire vive (RAM)** : 2 Go ou plus recommandés.
- **Connexion Internet** : Requise pour la première installation, les mises à jour et le téléchargement des nouvelles langues.

### 2. Autoriser l'Installation d'Applications Inconnues
Comme l'application est installée directement via un fichier APK (et non via le Play Store), Android peut bloquer l'installation par sécurité.
1. Allez dans les **Paramètres** de votre téléphone.
2. Cherchez **"Installer des applications inconnues"** ou **"Sources inconnues"**.
3. Autorisez votre navigateur (Chrome) ou votre gestionnaire de fichiers à installer des applications.

### 3. Google Play Protect
Lors de l'installation, un message peut dire "Application non reconnue".
- Cliquez sur **"Plus de détails"** puis sur **"Installer quand même"**.
*Note : Ce message est normal pour les applications qui ne sont pas encore sur le Play Store officiel.*

## 🌟 Fonctionnalités Principales
- **Hymnes & Louanges** : Accédez à vos recueils de chants préférés en plusieurs langues.
- **Journal d'Étude** : Prenez des notes textuelles, dessinez ou enregistrez des messages vocaux.
- **Mises à jour intégrées** : L'application vous prévient dès qu'une nouvelle version est disponible et peut l'installer directement.
- **Gestion intelligente du stockage** : Les fichiers joints (images, sons) sont automatiquement supprimés du téléphone quand vous supprimez une note.

## 🛠️ Support Technique
Si l'application ne s'ouvre pas ou se ferme toute seule :
1. Vérifiez que vous avez accordé les permissions nécessaires (Micro, Caméra, Stockage).
2. Essayez de vider le cache de l'application dans les paramètres Android.
3. Assurez-vous d'avoir assez d'espace libre sur votre téléphone.

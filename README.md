# AdvenTools

Une collection d'outils et de ressources pour la communauté adventiste du septième jour.

## 📁 Structure du projet

Ce projet contient plusieurs modules organisés par catégories :

### 📖 Bible (`/bible`)
- **20 versions bibliques** dans différentes langues
- Format SQLite3 pour une recherche efficace
- Langues disponibles : Français, Anglais, Malgache, Portugais, Espagnol, Allemand, Italien, et bien d'autres
- Version par défaut : Malagasy 1965

### 📚 Documents (`/docs`)
- **Esprit de Prophétie** : Œuvres d'Ellen G. White en malgache
- **Études Bibliques** : Guides d'étude et formations
- **Dossiers de l'Église** : Documents administratifs et organisationnels
- **Dossiers JA** : Ressources pour les jeunes adventistes
- **Séances de Prières** : Matériel pour réunions spéciales

### 🎵 Hymnes (`/hymnes`)
- **Cantiques Adventistes** en Français/Malgache
- Base de données de 600 KB
- Support pour recherche et affichage des paroles

### 🎨 Ressources (`/assets`)
- **Polices** : 16 polices pour l'interface
- **Images** : 14 ressources graphiques
- **Bibles** : Fichiers de données bibliques

## 🌟 Caractéristiques principales

- **Multilingue** : Support de nombreuses langues
- **Modulaire** : Architecture organisée par fonctionnalités
- **Léger** : Optimisé pour les appareils mobiles et faible bande passante
- **Accessible** : Interface intuitive pour tous les utilisateurs

## 🚀 Utilisation

### Pour les développeurs
```bash
# Cloner le dépôt
git clone https://github.com/Brayan-Clark/adventools.git

# Accéder au projet
cd adventools
```

### Structure des manifestes
Chaque module possède un `manifest.json` décrivant :
- Les versions disponibles
- Les métadonnées (taille, langue, URL)
- Les catégories et tags

## 📊 Statistiques

- **Bibles** : 20 versions (5.0 MB - 20.8 MB par version)
- **Documents** : 18 documents organisés en 5 catégories
- **Hymnes** : 1 collection de cantiques
- **Langues** : 15+ langues supportées
